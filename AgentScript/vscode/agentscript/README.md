# AgentScript VS Code Extension

This extension contributes:

- `.as` language registration
- syntax highlighting for AgentScript keywords, policies, strings, numbers, and call sites
- basic snippets for agents, workflows, and tools

To test locally:

1. Open this folder in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. Open an `.as` file such as `examples/legal_research.as`.
